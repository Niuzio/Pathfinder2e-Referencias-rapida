Pathfinder2e-referencia
==============

¡Descubre Pathfinder 2e Referencias Rápidas: tu pantalla de jugador web para Pathfinder Segunda Edición Remaster en español!

Reproduce la comodidad del PDF oficial de pantalla de jugador en formato web interactivo: acciones básicas, comunes y situacionales; acciones de habilidad; puntos de héroe; estados (como Acelerado o Tumbado); efectos ambientales (luz, terreno, sentidos); actividades de downtime, roles de exploración y especializaciones críticas/armadura. Todo accesible con un clic, sin abrir el manual en plena mesa.

Inspirado en el proyecto open-source dnd5e-referencia de cylira (licencia MIT), lo adapté para P2e con datos actualizados.

¡Pruébalo!

https://niuzio.github.io/Pathfinder2e-Referencias-rapida/
